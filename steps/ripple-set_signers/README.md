# ripple-set_signers

This step will setup signers on a accounts.

Example: parameters

```yaml
parameters:
  signers:
    - SignerEntry:
        Account: rsA2LpzuawewSBQXkiju3YQTMzW13pAAdW
        SignerWeight: 2
    - SignerEntry:
        Account: rsA2LpzuawewSBQXkiju3YQTMzW13pAAdW
        SignerWeight: 2
```
